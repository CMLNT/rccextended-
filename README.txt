rcc - стандартный компилятор ресурсов Qt, с одним неофициальным 
ключиком: -reverse. 
По данному ключу производится распаковка всех бинарных (*.rcc) файлов 
ресурсов в текущей директории. Распакованные ресурсы сортируются по 
отдельным папочкам, в соответствии с именем исходного rcc файла. 
Дополнительно генерируются файлы *.qrc и файл make.bat для обратной
сборки распакованных ресурсов в бинарные rcc файлы.
